Derivative

The following function that squares it's input 

    
def function(x):
    return x*x

If we ask by how much does the output change as we change the input we are deriving the derivative. If we change the vale of the input by just a very small bit that will be the rate of change over the small change. The following function does that 

    
def derivative(x):
    return(function(x+0.001)-function(x))/0.001

I have ploted out the results of the functions with the folloing pythonhe rate of change over the small change. The following function does that 

    
x=[]
y=[]
dy=[]
for i in range(21):
    x.append((i-10)/10)
    y.append(function((i-10)/10))
    dy.append((i-10)/10)   
plt.plot(x,y)
plt.plot(x,dy)
plt.show()

It can be seen that the derivative of x*x is 2x

If we use the derivative of the node output function with the error we will get a larger change in the weight the higher the error and much smaller change in the weight the lower the error. 

    
weight = 0.5
goal_pred = 0.8
input = 2
alpha = 0.1
w=[weight]
for iteration in range(20):
    pred  = input * weight
    derivative = input * (pred - goal_pred)
    weight = weight - (alpha * derivative)
    w.append(weight)
plt.plot(w)
plt.show()
